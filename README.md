 # Contribution Guide
 1. Clone the repo
 ```sh
 git clone https://github.com/dev-mkr/Focus-Hub.git
 ```
2. Install packages
 ```sh
   yarn install
   ```
3. Start local server
 ```sh
   npm run start 
   ```
4. Start tailwindcss 
 ```sh
   npm run watch:css
   ```
5. Add your changes
 ```sh
   git add *
   ```
6. Commit your changes 
 ```sh
   git commit -m "your commit messeage"
   ```
7. Push changes ðŸš€
 ```sh
   git push origin master
   ```
